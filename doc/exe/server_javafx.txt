pom.xml
=======

    <groupId>com.masanz</groupId>
    <artifactId>chat</artifactId>
    <version>1.0-SNAPSHOT</version>
    <name>chat</name>

    <properties>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
        <junit.version>5.9.2</junit.version>
        <main.class>com.aetxabao.chat.server.MainServer</main.class>
    </properties>


MANIFEST.MF
===========

Manifest-Version: 1.0
Main-Class: com.aetxabao.chat.server.MainServer


Maven
=====

mvn package


Mac .dmg installer
==================

/Library/Java/JavaVirtualMachines/jdk-17.jdk/Contents/Home/bin/jpackage --name ChatServer --icon src/main/resources/com/aetxabao/chat/server/app/icon.icns --input target --main-jar chat-1.0-SNAPSHOT.jar --main-class com.aetxabao.chat.server.MainServer


Win .exe installer
==================

"C:\Program Files\Java\jdk-17\bin\jpackage.exe" --name ChatServer --icon src/main/resources/com/aetxabao/chat/server/app/icon.ico --input target --main-jar ChatServer-1.0-SNAPSHOT.jar --main-class com.aetxabao.chat.server.MainServer
